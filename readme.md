# 📘 DANGAL — ICS Course Registration Planner

> A JavaFX-based course registration planner for students of the Institute of Computer Science (ICS), University of the Philippines Los Baños.

---

## 📌 Overview

**DANGAL** is a desktop application built with JavaFX that helps ICS students plan and organize their subjects for the **First Semester of Academic Year 2025–2026**. It provides a clean, interactive interface with a dynamic weekly schedule, a smart course-selection system, and conflict detection.

---

## ✨ Features

- **Program-Restricted Course Planning** — Only courses valid for your enrolled degree program can be added, preventing mismatched enlistments.
- **Visual Weekly Calendar** — Courses are automatically placed on the correct day and time slot in a color-coded weekly grid.
- **Complete Course Information** — View course code, title, description, units, schedule, instructor, and program before adding.
- **Automatic Conflict Detection** — The system checks for:
    - Overlapping time slots
    - Duplicate course entries
    - Courses outside the student's program
- **Multiple Schedule Setup** — Create differnt versions of your schedule
- **Schedule Comparison View** — Compare multiple schedule configurations side by side.
- **Profile Customization** — Students can upload a profile picture from their local machine.
- **Secure Login System** — Each student account stores credentials, degree program, and planned courses persistently.

## 🚀 How to Use

1. **Register** an account and select your degree program.
2. **Log in** to open your personalized dashboard.
3. Browse courses under the **Enlistment** tab for **1S AY 2025–2026**.
4. Click **Add Course** to enlist in a subject.
5. The course will automatically appear on the correct day and time on your **Calendar**.
6. Click any course block to view full details.
7. Use the **Compare Schedules** tab to view and compare different schedule setups.
8. Remove or edit courses anytime.
